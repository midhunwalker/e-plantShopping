# Paradise Nursery

A complete React 18 application with Redux Toolkit for managing a plant nursery e-commerce experience.

## Features

- **Landing Page**: Full-screen background with call-to-action
- **Product Listing**: Browse beautiful indoor plants
- **Shopping Cart**: Add, remove, and update quantities
- **Redux State Management**: Centralized cart state
- **React Router**: Seamless navigation
- **Responsive Design**: Mobile-friendly layout

## Tech Stack

- React 18.2.0
- Redux Toolkit 2.0.1
- React Redux 9.0.4
- React Router DOM 6.21.1
- React Scripts 5.0.1

## Installation

```bash
npm install
```

## Running the Application

```bash
npm start
```

The application will open at [http://localhost:3000](http://localhost:3000)
